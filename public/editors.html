Editors :

Editors : In general, an editor refers to any program capable of editing files. Some editors are notepad++, Sublime text Editors, VS code etc


How to download Sublime Editors ?


Step 1 : Goto Google type on Search bar www.sublimetext.com

Step 2 : Click on Download for Windows.


How to install Sublimetext ?


Step 1: Right click on Sublime text and Click on Run as administrator

Step 2: Click on Yes-> Next-> install-> Finish.


Some important points to remember in Sublimetext Editor :

1) Set your file as html: Goto View tab -> Syntax -> HTML.

2) To Generate dummy text in sublime : Type Lorem and Press Tab Button

3) To Generate complete HTML code : Type <html then press Enter.
